# BGM Ethiopia pilot — evidence pack (public)

This repository contains a public, non-turnkey **technical evidence pack** (results + specification) for a dyadic **household → camp** destination choice model.

## Read the evidence pack
- **ETHIOPIA_TECH_EVIDENCE_PACK.md**

## What this is (and is not)
- Includes: model specification (no OD-pair memorization), headline AUCs, robustness table, and interpretability snippets.
- Does not include: raw microdata, full feature engineering pipeline, or reproducible end-to-end code.
